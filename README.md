## üöÄ Installing boa-hancock-api

#### Clone repository

```powershell
  git clone https://github.com/RodrigoRVSN/boa-hancock-api.git
```

#### Install dependencies

```powershell
  pnpm i
```

#### Run server

```powershell
  pnpm start:dev
```

#### Open prisma studio to access and alter database

```powershell
pnpm prisma studio
```
